# Transportation Network System

A Java Swing-based desktop application for modeling and analyzing transportation networks using graph algorithms. The project provides an interactive visualization of cities and roads while demonstrating shortest path and minimum spanning tree computations.

## Features

* Interactive transportation network visualization
* Add cities and roads with weighted distances
* Shortest Path using Dijkstra's Algorithm
* Minimum Spanning Tree using Prim's Algorithm
* Graphical User Interface built with Java Swing

## Technologies Used

* Java
* Java Swing
* Graph Data Structures
* Dijkstra's Algorithm
* Prim's Algorithm
* Object-Oriented Programming (OOP)

## Project Structure

* `Graph.java` – Graph implementation and algorithms
* `NetworkPanel.java` – GUI visualization of the transportation network
* `Main.java` – Application entry point and user interface

## How to Run

1. Ensure Java JDK is installed.
2. Open the project folder in the terminal.
3. Compile the project:

```bash
javac *.java
```

4. Run the application:

```bash
java Main
```

This project demonstrates:

* Graph Data Structures
* Shortest Path Algorithms
* Minimum Spanning Tree Algorithms
* Java Swing GUI Development
* Object-Oriented Programming
